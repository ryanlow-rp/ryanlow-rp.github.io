PythonAnywhere

PythonAnywhere is an online integrated development environment (IDE) and web hosting service (Platform as a service) based on the Python programming language.

Flask

Flask is a micro web framework written in Python. This means Flask provides you with tools, libraries and technologies that allow you to build a web application.

Quick start guide (https://flask.palletsprojects.com/en/2.0.x/quickstart/)

  1. Go to www.pythonanywhere.com and click on Start running Python online in less than a minute

  1. Click on Create a Beginner account

  1. Enter your account information, check the checkbox and click on Register

  1. Click on Web

  1. Click on Add a new web app

  1. Click on Next

  1. Click on Flask

  1. Click on Python 3.9 (Flask 2.0.0)

  1. Click on Next

  1. The configuration for your web app is available on this page. Your webapp URL is yourusername.pythonanywhere.com. Open this URL in a new tab and you should see a page with the text Hello from Flask!.

  1. On the web app configuration page, scroll down to the Code section and change the working directory to the same value as your source code directory. To edit the value for the Working directory, click on the associated value, modify it and click on the checkmark button.
  2. You can view the log files via the Log files section.

  1. Click on Files to view your files. The mysite directory contains the files for your web app. Click on mysite.

  1. The mysite folder contains a single flask_app.py file.
