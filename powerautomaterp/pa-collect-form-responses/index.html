What you'll build

In this codelab, you will create an automated cloud flow.

The flow will trigger when a form is submitted.

The flow will collect responses from the form and save the responses to an Excel file

What you'll learn

  • How to create an automated cloud flow
  • How to create a simple form
  • How to read Form response
  • How to write to Excel

What you'll need

  • A web browser
  • Access to Forms, OneDrive, Power Automate
  1. Launch Microsoft OneDrive
  2. In the folder PowerAutomateRP, create a file responses.xlsx
  3. Set cell A1 as email
  4. Set cell A2 as response
  5. Click on cell A1
  6. Click on Format As Table and choose a style
  7. Check the range of cells to be formatted as a table is correct
  8. Ensure that My table has headers is checked

  1. Click on cell A1
  2. Click on Table Design
  3. Note that the name of the table is Table1
  1. Launch Microsoft Forms
  2. Click on New Form

  1. Set the form title as Collect responses
  2. Add a Text question type
  3. Set the question as Enter your response
  4. Toggle Required on

  1. Click Collect responses
  2. Check Shorten URL
  3. Click Copy link and save that link somewhere for later
  1. Launch Microsoft Power Automate

Create the flow

  1. Click Create
  2. Click Automated cloud flow

  1. Set the flow name as Collect responses
  2. Select When a new response is submitted
  3. Click Create

Turn off the new designer

  1. Toggle the New designer to off

Select the form to trigger the flow

  1. For the Form Id, select the form Collect responses

  1. Add an action Get response details
  2. For the Form Id, select the form Collect responses
  3. For the Response Id, select the dynamic content Response Id

  1. Save and test. The flow is waiting for a form submission.

  1. Using the form link you copied earlier, submit a response to your form
  2. Expand Get response details to see the run result

  1. Resume editing the flow
  2. Add the action Add a row into a table
  3. Set the field values based on the table
