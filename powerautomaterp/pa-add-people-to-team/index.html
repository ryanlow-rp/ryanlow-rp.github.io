Core Concepts

A Power Automate flow consists of 1 trigger and 1 or more actions

  • Instant cloud flow - triggered manually
  • Automated cloud flow - triggered by a connector
  • Scheduled cloud flow - triggered based on a predetermined schedule

Connector

Connectors serve as the bridge between Power Automate and external services

Connectors offer triggers and actions:

  • Triggers - Events that start a workflow
  • Actions - Tasks carried out once a workflow has started
